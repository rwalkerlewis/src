Synopsis
        sftracealign < infile.rsf monitor=monitorfile.rsf > outfile.rsf times=timefile.rsf maxshift= nlen= vel=1500.

 
Parameters
       
 
maxshift= 		Maximum allowed time shift
 
file monitor= 		auxiliary input file name
 
nlen= 		Window length of shift vector (in samples)
 
file times= 		auxiliary output file name
 
vel=1500. 		Rupture speed for linear shift
