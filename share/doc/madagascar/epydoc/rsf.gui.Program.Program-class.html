Class Program

source code

A Program is just a wrapper for an RSF Program, that contains all of the information from the RSF self-documentation construct.

Ideally, this would be replaced with the rsfprog class from rsf.doc but that class is not designed to be used with this, and has a lot of extra methods that are not necessary.
