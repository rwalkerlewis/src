Class Sandbox

source code

The Sandbox class contains the Canvas where all of the graphical action happens.

Sandboxes are responsible for keeping track of Flows attached to them, and allow users to decide how to interact with Flows.

A Sandbox must be attached to a Toplevel or root widget.
