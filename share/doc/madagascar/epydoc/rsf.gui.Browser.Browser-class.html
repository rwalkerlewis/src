Class Browser

source code

The Browser is a Frame that contains all the searching, buttons and other tools used to quickly browse and search for Madagascar programs.

If it is attached to a Canvas, then it can AddFlows directly to that Canvas.

Otherwise, it can standAlone, meaning that it will exclude certain features, such as Buttons to add items to a Canvas.

To run in standalone mode directly call this program from the command line.
